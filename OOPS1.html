Object-oriented programming(oops) is a programing padradigm that uses objects, which can contain data in the form of fields(often known as properities or attributes)and code in the form of procedured(often known methods). javascript is a versatile language that supports OOP principles through its prototype based inheritance syatem.

1. CLASS: a Class in javascript is a blueprint for creating objects. it defines set of proprerities and methods created from the class will have.

2.Object: obj in JS are cpmplex data types that can store data have asspciated behaviour through properties nd methods.

3. Inheritance: is a mechanism that allows a class (subclass/derived class)to inherit properitites and methods from another class(superclass/base class).Its promotes code reuse and establishes an "is-a" lationship between classes.

4. Polyorphism: it allows objects of different classes to be trated as object of common base class.
